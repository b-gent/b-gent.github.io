Description

PKWARE data compression library (DCL).

PKWARE DCL ("explode") ("PKWARE data compression library") decompressor used in engines:

  • AGOS (exclusively for Simon 2 setup.shr file)
  • Mohawk
  • Neverhood
  • SCI
