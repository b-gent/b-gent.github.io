42 /* Experimental code for emulation of the chip's busy flag wait cycle.
43  * Explanation:
44  * Before attempting a port write a client application would usually read the chip's
45  * busy flag and remain in a loop until the flag is cleared. This does not work with
46  * an emulator that is on the same thread as the client code (the busy flag will never
47  * clear). Instead, I emulate a wait cycle by withholding (enqueueing) incoming register
48  * writes for the duration of the wait cycle.
49  * For now I have disabled this with an #ifdef since I haven't seen any impact on the
50  * sound.
51  */
