131 /*
132  * The classes below correspond to the interfaces defined in c_types.h and provided for convenience when using C++.
133  * The approach used makes no assumption of any internal class data memory layout, since the C++ standard does not
134  * provide any detail in this area and leaves it up to the implementation. Therefore, this way portability is guaranteed,
135  * despite the implementation may be a little inefficient.
136  * See c_types.h and c_interface.h for description of the corresponding interface methods.
137  */
