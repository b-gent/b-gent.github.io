231 /* Enqueues a MIDI event for subsequent playback.
232  * The MIDI event will be processed not before the specified timestamp.
233  * The timestamp is measured as the global rendered sample count since the synth was created (at the native sample rate 32000 Hz).
234  * The minimum delay involves emulation of the delay introduced while the event is transferred via MIDI interface
235  * and emulation of the MCU busy-loop while it frees partials for use by a new Poly.
236  * Calls from multiple threads must be synchronised, although, no synchronisation is required with the rendering thread.
237  * onMIDIQueueOverflow callback is invoked when the MIDI event queue is full and the message cannot be enqueued.
238  */
