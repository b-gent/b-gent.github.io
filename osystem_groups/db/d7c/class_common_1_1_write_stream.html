◆ finalize()

virtual void Common::WriteStream::finalize ( )
inlinevirtual

Finalize and close this stream.

Call this method right before this stream instance is deleted. The goal is to enable calling the code to detect and handle I/O errors that might occur when closing (and flushing, if buffered) the stream.

After this method has been called, no further writes can be performed on the stream. Calling err() is allowed.

By default, this just flushes the stream.
