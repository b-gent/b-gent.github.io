◆ blitOffscreenBuffer()

void TinyGL::FrameBuffer::blitOffscreenBuffer ( Buffer *  buffer)

Blit the buffer to the screen buffer, checking the depth of the pixels. Eack pixel is copied if and only if its depth value is bigger than the depth value of the screen pixel, so if it is 'above'.
