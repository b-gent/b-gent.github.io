*AES-XTS - AES Xor-encrypt-xor with ciphertext stealing mode*

Type: ``bool``

Help
====

Enable the AES Xor-encrypt-xor with ciphertext stealing mode (XTS) mode,
MBEDTLS_CIPHER_MODE_XTS setting in mbed TLS config file.
