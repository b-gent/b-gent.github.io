.. title:: NRF_SECURITY_ADVANCED

.. option:: CONFIG_NRF_SECURITY_ADVANCED

*Advanced mbed TLS Configuration Settings*

Type: ``bool``

Help
====

This setting will enable the advanced configuration menu.
The advanced configuration allows for further fine tuning of the mbed TLS configuration
by adjusting , as example: SSL maximum content sizes, disabling of specific cipher suites,
ECP bit sizes, Bignum options.

Direct dependencies
===================

\ :option:`MBEDTLS_VANILLA_BACKEND <CONFIG_MBEDTLS_VANILLA_BACKEND>` && \ :option:`MBEDTLS_VANILLA_BACKEND <CONFIG_MBEDTLS_VANILLA_BACKEND>` && \ :option:`NORDIC_SECURITY_BACKEND <CONFIG_NORDIC_SECURITY_BACKEND>`

*(Includes any dependencies from ifs and menus.)*

Defaults
========

No defaults. Implicitly defaults to ``n``.
