The SoftDevice Controller is an RTOS-agnostic library built for the Nordic Semiconductor nRF52 and nRF53 Series that supports Bluetooth 5. In addition to providing a Bluetooth HCI interface, it also provides:

  • Multiprotocol support through APIs for requesting RADIO timeslots

  • APIs for accessing flash memory concurrently with radio activity

  • APIs for controlling the high frequency clock

The SoftDevice Controller is distributed in different variants containing different features. All variants are available as soft-float, softfp-float, and hard-float builds.
