Detailed Description

Simple random number generator. Although it is definitely not suitable for cryptographic purposes, it serves our purposes just fine.

Common::RandomSource::RandomSource ( const String &  name)

Construct a new randomness source with the specific name. The name used name must be globally unique, and is used to register the randomness source with the active event recorder, if any.
