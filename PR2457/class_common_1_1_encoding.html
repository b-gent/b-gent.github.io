Detailed Description

A class, that allows conversion between different text encoding, the encodings available depend on the current backend and if the ScummVM is compiled with or without iconv.

char* Common::Encoding::convert ( const char *  string,
size_t  length

Converts string between encodings. The resulting string is ended by a character with value 0 (C-like ending for 1 byte per character encodings, 2 zero bytes for UTF-16, 4 zero bytes for UTF-32)

The result has to be freed after use.

Parameters
stringString that should be converted.
lengthLength of the string to convert in bytes.
Returns
Converted string (must be freed) or nullptr if the conversion failed
