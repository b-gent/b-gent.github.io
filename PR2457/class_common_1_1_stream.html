Detailed Description

Virtual base class for both ReadStream and WriteStream.

Member Function Documentation

◆ err()

virtual bool Common::Stream::err ( ) const
inlinevirtual

Returns true if an I/O failure occurred. This flag is never cleared automatically. In order to clear it, client code has to call clearErr() explicitly.

Note
The semantics of any implementation of this method are supposed to match those of ISO C ferror().

Reimplemented in Common::DumpFile, Common::MemoryWriteStream, Common::File, Common::SubReadStream, and Common::OutSaveFile.

◆ clearErr()

virtual void Common::Stream::clearErr ( )
inlinevirtual

Reset the I/O error status as returned by err(). For a ReadStream, also reset the end-of-stream status returned by eos().

Note
The semantics of any implementation of this method are supposed to match those of ISO C clearerr().

Reimplemented in Common::MemoryReadWriteStream, Common::DumpFile, Common::MemoryWriteStream, Common::File, Common::MemoryReadStream, Common::SubReadStream, and Common::OutSaveFile.
