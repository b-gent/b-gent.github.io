Detailed Description

Wrapper class which holds a pointer to a pointer to a CoroBaseContext. The interesting part is the destructor, which kills the context being held, but ONLY if the _sleep val of that context is zero. This way, a coroutine can just 'return' w/o having to worry about freeing the allocated context (in Simon Tatham's original code, one had to use a special macro to return from a coroutine).
