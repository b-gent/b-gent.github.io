Detailed Description

template<class Key, class Val, class HashFunc = Hash<Key>, class EqualFunc = EqualTo<Key>>
class Common::HashMap< Key, Val, HashFunc, EqualFunc >

HashMap<Key,Val> maps objects of type Key to objects of type Val. For each used Key type, we need an "size_type hashit(Key,size_type)" function that computes a hash for the given Key object and returns it as an an integer from 0 to hashsize-1, and also an "equality functor". that returns true if if its two arguments are to be considered equal. Also, we assume that "=" works on Val objects for assignment.

If aa is an HashMap<Key,Val>, then space is allocated each time aa[key] is referenced, for a new key. If the object is const, then an assertion is triggered instead. Hence if you are not sure whether a key is contained in the map, use contains() first to check for its presence.
